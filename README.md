# TM pro-english

TM pro-english is a modern, responsive EdTech landing/application prototype for learning English. It includes progressive levels, a global progress system, interactive exercises, a 0–1000 number module, an AI-style practice chat, an integrated translator demo, badges/statistics, and light/dark mode.

## Run locally

Open `index.html` directly in a browser, or serve the folder with any static web server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
